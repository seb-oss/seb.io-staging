Components

Breadcrumb

A breadcrumb is a secondary navigation showing the website hierarchy.

Button

Buttons are an essential element of interaction design. Their primary role is to help people take actions quickly.

Checkbox

A checkbox is a form element that allows users to select one or multiple options from a list.

Input

Text fields allow users to enter text into a UI. They typically appear in forms and dialogs.

Radiobutton

A radio button allows users to select one option from a predefined set of mutually exclusive choices.
